## Running the project

### Run Expo

**Install dependencies**

```
npx / yarn start
```

**Install IOS Pods**

### Run Native Development

**Install dependencies**

```
yarn
```

**Install IOS Pods**

```
cd ios && pod install
```

**Run IOS**

```
yarn ios
```

**Run Android**

```
yarn android
```
